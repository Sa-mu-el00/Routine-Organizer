# Routine-Organizer (English)
Axiomatic Dashboard: A high-performance productivity tool.  
Interactive Calendar: Manage routines and events. 
Intelligent Logic: Supports recurring and specific tasks. 
Analytics: Monitors daily performance and statistics.  
Technology: HTML5, Tailwind CSS, JS (ES6+) and LocalStorage.  
User Experience: Dark mode and responsive design.
